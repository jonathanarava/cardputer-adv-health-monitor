# cardputer-adv-health-monitor
Cardputer ADV S3 version Hardware Debugger (GPIO, on-board sensors)

Steps to run:
1. Build platform.io project with the source code. 3 files: device config, main, ux/ui
2. Connect Cardputer to computer over type-C
3. Click the "upload" botton on vscode (bottom bar).
4. Open the html file (integrated or browser) and click "Connect". This will connect to cardputer over COM4 (default serial port) and start rendering data.
